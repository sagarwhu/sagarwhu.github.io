Project Deliverables:

The rough timeline for various deliverable are as follows:

1.          Prepare project proposal by creating vision to summary your project (Week 1)

2.          Choose four use cases (each use case should include those different scenarios), provide full-dressed descriptions. Each scenario should discuss a different use of your system. (Week 2)

3.          Prepare your project structure by using bootstrap framework (Week 3)

4.          Project Demo Presentation (Week 4)

5.          Create system sequence diagram, activity diagram and domain model of your project (Week 5)

6.          Submission of project design documentation (Week 6)

7.          Create class and sequence diagram of your project according to use case diagram (Week 7)

8.          Create ER diagram and present the project database design (Week 8)

9.          Develop the four case cases functionally (Week 9)

10.    Project performance1 (Week 10)

11.    Project performance1 (Week 11)

12.    Project performance1 (Week 12)

13.    Project performance1 (Week 13)

14.    Project performance1 (Week 14)

15.    Project Final Presentation and provide design documentation with user manual (Week 15 and 16)
